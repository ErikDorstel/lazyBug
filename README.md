# lazyBug
ESP32 based Hexapod Robot
#### GPIO
* GPIO 21 - SDA PCA9685
* GPIO 22 - SCL PCA9685
* GPIO 17 - SDA MPU-6050 and VL53L0X
* GPIO 16 - SCL MPU-6050 and VL53L0X
* GPIO 18 - INT VL53L0X (not in use)
* GPIO 19 - XSHUT VL53L0X
#### I2C Adresses
* PCA9685 - 0x40 and 0x41
* MPU-6050 - 0x68
* VL53L0X - 0x29
